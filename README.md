# NEXUS Engineering & Technology Website

Professional static website for NEXUS Engineering & Technology.

## Focus Areas

- IoT platforms
- Automation solutions
- Automotive systems
- ROV systems

## Files

- `index.html` - Main website page
- `styles.css` - Responsive visual design
- `script.js` - Arabic/English language toggle
- `assets/nexus-logo-cropped.svg` - NEXUS logo asset

The site can run directly by opening `index.html` in a browser.
